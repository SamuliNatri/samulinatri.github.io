Description

In the first part of the book we create a todo app and deploy it to Heroku. It introduces a beginner to all essential Django features. The focus is on fundamentals, not on the project structure or frontend.

In the second part we create a website for a pizzeria. This time we pay more attention to the structure of the project. It introduces another level of abstraction by utilizing Django's class-based views. It shows how to work with images and serve static files from Amazon S3 storage (through Amazon's content delivery network). Bootstrap JavaScript framework is used for the frontend.
